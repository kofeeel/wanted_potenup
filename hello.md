## Branch 명령어어
- 브랜치(branch): 독립적인 작업 공간
- 머지(merge): 브랜치 간 작업 통합
- 체크아웃(checkout): 특정 브랜치 또는 커밋으로 이동

## 8. pull 및 충돌 해결하기 

- pull = fetch + merge
- 서버의 내용이 최신일 경우 pull 적용
- 충돌 원인: 동일 파일 편집 시 주로 발생
- 해결 방법:
    - 에디터 직접 수정
